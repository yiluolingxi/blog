## 1. 前提提要、场景

`Linux` 三剑客（`grep`、`sed`、`awk`）之一。      

`awk` 是一个文本分析工具。具有以下特点:
- 截取文档中的某个段。
- 像 `sed` 一样过滤出匹配字符的行。
- 有条件操作符
- 有内置变量


## 2. awk
命令：`awk [ -F fs ] 'prog' [ file ... ]`      
- `-F`：设置分隔符，后面跟随 `fs`（`field separator`）分隔符
- `prog`: `awk` 语法，或者称为 `Action`

示例：`awk '{print $0}' demo.txt`


## 3. Action 动作
`awk` 中要执行的语句或命令称为 `Action`，即 `prog`。
