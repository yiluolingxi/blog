## 2. 初步了解
全称 **Hypertext Transfer Protocol Secure** ，中文名 **超文本传输安全协议**。在HTTP和TCP中间添加一个加密/身份验证层（SSL层），可以理解是HTTP的安全升级版。   
HTTPS的目的就是为了去除中间人，避免中间人攻击。



## 3. 整体框架
对称加密 + 非对称加密 + CA证书     
利用了以上几项结合去实现HTTPS加密机制。接下来会逐步讲解，指明各项优缺点，以及为何需要结合使用。
